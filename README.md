## About Me

I like factory games. I like automating things. One day I wondered: "Can I make money with this?"

Turns out, yes. Data engineering is just building factories for data — miners, conveyor belts, storage containers, the whole thing. The same mindset that optimizes a Satisfactory production line applies directly to building data pipelines.

I have a CS degree and previous QA experience, but I'd never touched data engineering until recently. Two years of career uncertainty. Then I asked the right question and everything clicked.

Each project has a detailed README explaining the architecture, setup, and my thought process. I treat my portfolio like I would treat a production system.

The factory is open.
